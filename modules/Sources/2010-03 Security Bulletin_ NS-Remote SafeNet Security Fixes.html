2010-03 Security Bulletin: NS-Remote SafeNet Security Fixes

JSA10429
2010-03-01
2013-05-09
All NetScreen-Remote versions prior to 9.0R5
High
7.0

This Security Bulletin addresses security flaws that have been fixed in NS-Remote 9.0r5:
- SafeNet SoftRemote Policy File Handling Remote Buffer Overflow Vulnerabilities (Juniper reference KB 36907, SafeNet Bulletin 103009-1)
- SafeNet SoftRemote IKE Service Remote Stack Buffer Overflow Vulnerability (Juniper reference KB 35154, SafeNet reference QA62358)

Upgrade to NS-Remote 9.0r5 or later is recommended.

No workarounds are available.
